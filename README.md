 - Tailwind CSS

## Run Locally

Clone the project

```bash
  git clone https://github.com/Sudcsk24/DefiHub.git
```


## Install dependencies
install with npm

```bash
  npm install
```
yarn install
```bash
  yarn install
```

Start the server

```bash
  npm run dev
```
